Keynga

The King of Keys

Features About Project

A one type password manager and generator aiming at efficiently managing platform keys in a safe robust way

Save the hassle of remembering tonnes of passwords for the various platforms, and save them in your personal vault, encrypted with modern day cipher algorithms, with ONE MASTER PASSWORD

Features

Secure Platform and Password Management

Add Platforms along with their usernames and passwords. The passwords are encrypted and stored securely. Generate strong, random passwords for platforms, enhancing security. View and Copy Passwords stored in one’s vault, with the option to copy passwords to the clipboard for easy access.

User Authentication and Profile Management

Registration and Login with a username and master password, and subsequently log in securely. The master password is used to encrypt and decrypt user data. Profile Viewing: Users can view their profile information, including their unique user ID and the list of platforms they have added.

Platform Management

Manage Platforms by updating existing platform information, including changing passwords. Delete Platforms no longer in use, ensuring that old or unused accounts do not pose a security risk.
